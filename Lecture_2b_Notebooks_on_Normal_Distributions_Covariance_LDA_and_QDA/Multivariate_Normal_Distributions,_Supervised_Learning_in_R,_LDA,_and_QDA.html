Create a linear classifier that uses the vector w to define a "discriminant": a boundary between two classes.¶

In this case w defines a "direction" vector between the two classes, and the discriminant is orthogonal to it. We also allow a constant c to shift the discriminant back and forth between the classes

classifier = function(x, w, c) { 2 * ((x %*% w + c) > 0) - 1 }
                             # i.e., ((x * w + c) > 0) ? +1 : -1
